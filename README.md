# Christianity by Country Dashboard (Tableau)

## 📊 Overview

This project analyzes the global distribution of Christianity using country-level data and interactive visualizations.

The workflow includes data cleaning, transformation, and visualization, demonstrating a complete data analysis pipeline from raw dataset to dashboard insights.

---

## 🎯 Objectives

* Analyze the percentage of Christians by country
* Identify regions with the highest and lowest concentrations
* Transform raw data into a structured, analysis-ready format
* Visualize global patterns using an interactive dashboard

---

## 🛠️ Tools & Technologies

* Microsoft Excel / CSV (Data Cleaning & Preparation)
* Tableau Public (Data Visualization)

---

## 📁 Dataset

The dataset includes:

* Country
* Total population
* Christian population
* Percentage of Christians
* Additional demographic indicators

---

## 🧹 Data Cleaning & Preparation

Before visualization, the dataset was cleaned and transformed to ensure accuracy and consistency:

* Removed duplicates and irrelevant fields
* Standardized country names for proper geographic mapping
* Handled missing/null values
* Ensured correct data types (numeric vs categorical)
* Prepared fields for Tableau compatibility

This step ensured the dataset was reliable and ready for visualization.

---

## 📊 Dashboard Components

### 🌍 Christianity Map

* Choropleth map showing % of Christians by country
* Highlights global and regional distribution patterns

### 📊 Bar Chart (Country Comparison)

* Displays percentage of Christians by country
* Enables direct comparison across nations

### 🧩 Treemap (Regional Distribution)

* Visualizes distribution by continent/region
* Emphasizes relative size and contribution

## 📈 Key Insights

* Christianity is highly concentrated in Europe, the Americas, and parts of Africa
* Countries with large populations do not always have the highest percentage representation
* Regional differences highlight cultural and historical influences
* Combining absolute and relative metrics provides deeper analytical insight
